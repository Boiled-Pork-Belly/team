# ✅ 팀원 소개 웹사이트 QA 체크리스트

웹사이트 배포 전 확인해야 할 핵심 테스트 항목입니다.

## 1. 데이터 표시 (Data Display)
- [ ] **팀원 목록 조회**: 저장된 모든 팀원이 빠짐없이 목록에 표시되는가?
- [ ] **필수 정보 확인**: 이름, 역할, MBTI 등 핵심 정보가 누락 없이 출력되는가?
- [ ] **이미지 처리**: 프로필 이미지가 없는 경우 깨지지 않고 기본 이미지로 대체되는가?
- [ ] **링크 이동**: SNS, 블로그 등 외부 링크 클릭 시 새 탭으로 정상 연결되는가?
- [ ] **줄바꿈 처리**: 자기소개 텍스트가 길어질 경우 레이아웃을 해치지 않고 적절히 줄바꿈 되는가?

## 2. 기능 동작 (Functionality)
- [ ] **검색 기능**: 이름, 역할 등으로 검색 시 실시간으로 결과가 필터링되는가?
- [ ] **결과 없음 처리**: 검색 결과가 없을 때 안내 메시지(예: "검색 결과가 없습니다")가 표시되는가?
- [ ] **상세 모달/페이지**: 팀원 클릭 시 상세 정보 창이 정상적으로 열리고 닫히는가?

## 3. 사용자 경험 (UX & UI)
- [ ] **반응형 레이아웃**: 모바일, 태블릿, PC 화면 크기에 맞춰 배치가 자연스럽게 조정되는가?
- [ ] **로딩 피드백**: 데이터를 불러오는 중일 때 로딩 애니메이션이 표시되는가?
- [ ] **에러 피드백**: 데이터 로드 실패 시 사용자에게 적절한 안내(예: "다시 시도해주세요")가 뜨는가?
- [ ] **호버 효과**: 버튼이나 카드에 마우스를 올렸을 때 클릭 가능하다는 시각적 효과가 있는가?
